Comencemos

Este proyecto lo desarrollé utilizando Angular 19, con el objetivo de crear una plataforma eficiente y sencilla para la gestión de mantenimientos de empresas y colaboradores. La idea era facilitar el seguimiento y control de las empresas, sus colaboradores y los mantenimientos que realizan. Así, logré crear una solución práctica, donde los usuarios pudieran gestionar de manera sencilla todos estos aspectos.

Diseño

Tecnologías Utilizadas

Para garantizar una solución moderna y escalable, utilicé las siguientes tecnologías:

  • Angular 19: El framework que utilicé para construir la aplicación, proporcionando una estructura sólida y escalabilidad.
  • Firebase: Elegí esta base de datos en tiempo real para almacenar los datos de empresas, colaboradores y mantenimientos, lo que me permitió realizar la sincronización en tiempo real.
  • Flowbite: Utilicé esta biblioteca de componentes basada en Tailwind CSS para desarrollar una interfaz moderna, eficiente y completamente responsiva.
  • HTML y CSS: Son los lenguajes de marcado y estilo que empleé para la estructura y el diseño de la página, asegurando que la experiencia de usuario fuera atractiva y fluida.

Material de Apoyo

Para desarrollar este proyecto, utilicé la documentación oficial de Angular 19, lo que me permitió enriquecer mis conocimientos y aplicar mejores prácticas en el desarrollo. También empleé Firebase como base de datos en modo de prueba, implementando un CRUD completo para gestionar diversas entidades:

  • Países
  • Departamentos
  • Municipios y sus respectivas dependencias
  • Empresas y Colaboradores

Este material de apoyo fue clave para el desarrollo, ya que me permitió implementar funcionalidades cruciales, como la gestión de datos en tiempo real y la administración eficiente de las entidades mencionadas.
